<h2>Интерфейсы</h2>

<h3>Теория:</h3>

<ul>
    <li>
        <pre><code>
            type iface struct {
                tab *itab
                data unsafe.Pointer
            }
        </code></pre>
    </li>
    <li>
        Интерфейсы - абстракция над типом. Состоит из ссылки на тип данных и на сами данные.
        Интерфейсы используются для уменьшения связанности кода (decoupling).
        Интерфейс должен быть объявлен в месте использования (SOLID).
    </li>
    <li>
        Размер пустого интерфейса = размеру 2 ссылок. В 64-битных системах - 2  * 8 = 16 байт, в 32-битных - 2 * 4 = 8 байт.
    </li>
    <li>
        Переменная типа interface{}/any (пустой интерфейс) будет равна nil, если обе ссылки равны nil, то есть не указан ни тип, ни данные.
    </li>
</ul>
